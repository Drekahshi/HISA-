# **App Name**: Veridi

## Core Features:

- Data Submission: Allow validators to send tree planting records using a POST request.
- Record Hashing: Hash tree planting records upon submission.
- Blockchain Anchoring: Store record hashes, along with validator identity, on the Hedera Consensus Service.
- AI Validation Tool: Use an AI agent tool that checks the health of the registered tree and report health, or if they are affected by any natural phenomenon, such as storms and fires.
- JANI Balance Tool: Allow AI agents to check JANI token balances.
- Impact Dashboard: Display a real-time dashboard for tracking verified trees and JANI token distribution.
- Mobile App: Mobile application for conservation and enviromental regeneration efforts

## Style Guidelines:

- Primary color: Forest green (#3F704F) to represent nature and growth.
- Background color: Light beige (#F5F5DC) to provide a natural, earthy feel.
- Accent color: Earthy brown (#8B4513) to highlight important elements.
- Headline font: 'Alegreya', a humanist serif with an elegant, intellectual, contemporary feel.
- Body font: 'Inter', a grotesque-style sans-serif with a modern, machined, objective, neutral look.
- Use minimalist icons that visually represent conservation efforts, tree planting, and environmental regeneration.
- Employ subtle animations on the dashboard to show the growth and impact of the conservation efforts.
